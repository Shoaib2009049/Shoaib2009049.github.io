# Software Engineer Portfolio

A modern, responsive portfolio website built with HTML, CSS (Tailwind), and JavaScript.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Elements**: Hover effects, smooth scrolling, and animated skill bars
- **Performance Optimized**: Fast loading with CDN-based resources
- **Accessible**: Semantic HTML and proper contrast ratios

## Sections

1. **Hero Section**: Eye-catching introduction with call-to-action buttons
2. **About Me**: Personal introduction with skills showcase
3. **Projects**: Featured projects with technology tags and links
4. **Contact**: Multiple ways to get in touch

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- Lucide Icons
- Google Fonts (Inter)

## Setup

1. Open `index.html` in any modern web browser
2. No build process required - everything runs client-side
3. For development, use a local server like Live Server in VS Code

## Customization

### Personal Information
- Update name, title, and bio in the HTML
- Replace placeholder email and social links
- Add your own project details and images

### Styling
- Modify colors in Tailwind classes
- Add custom CSS in `styles.css`
- Adjust animations and transitions

### Content
- Add more projects to the projects section
- Update skills and percentages
- Customize the about section with your experience

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Performance

- Optimized images and assets
- Minimal JavaScript for fast loading
- CDN-based resources for better caching
- Responsive images for different screen sizes
